--- title: Stable build 0.1.2-23 time: 09:14:36 date: February 12, 2013 category: news summary: Stable build 0.1.2-23 is out --- New terracoin stable release 0.1.2-23 is now available, pre-built windows binaries are available sourceforge. This build includes a lot of performance improvements, along with a set of new features. New rpc commands are now available, to lock/unlock unspent "coins", importprivkey rpc command was also enhanced to enable/disable blockchain rescan, [terracoin transaction notifier](https://github.com/terracoin/terracoin/wiki/TransactionNotifier) is now part of stable builds.
