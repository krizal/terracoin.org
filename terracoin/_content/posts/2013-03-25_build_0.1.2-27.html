--- title: Stable build 0.1.2-27 time: 10:14:36 date: March 25, 2013 category: news summary: New terracoin client build 0.1.2-27 is available for download --- This build includes multiple performance-related improvements, a few core enhancements and various desktop client tweaks. Our transaction notifier now automatically sign the transmitted json document (see the transaction notifier wiki page for more informations). New seednodes were added, and the IRC bootstrap mechanism was re-enabled, to protect from a seednode attack.
