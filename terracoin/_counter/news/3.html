Stable build 0.1.2-23

February 12, 2013 | news

New terracoin stable release 0.1.2-23 is now available, pre-built windows binaries are available sourceforge.

This build includes a lot of performance improvements, along with a set of new features.

New rpc commands are now available, to lock/unlock unspent “coins”, importprivkey rpc command was also enhanced to enable/disable blockchain rescan, terracoin transaction notifier is now part of stable builds.

New transaction notifier feature

December 23, 2012 | news

First implementation/iteration of TerraCoin “TransactionNotifier” feature was recently pushed to the “dev-0.1.2” development branch, and a win32 binary package was uploaded to sourceforge.

Described here, this feature provides the terracoin application the ability to notify a remote http server of incoming transactions.

While beeing quite simple right now, this could allow merchant (or other) applications to run without the need for a local terracoin daemon, running one or more terracoin daemons at distant locations.

p2pool now supports terracoin

December 8, 2012 | news

we submitted a patch to p2pool to provide terracoin support.

Thanks to p2pool author, those modifications were quickly merged into p2pool official sourcecode.

This gives you another pool-mining environment, so you can generate terracoins using a decentralized, DoS-resistant mining pool. Our support pages should soon provide detailed instructions, until then, here’s a quick p2pool howto.
